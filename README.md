# Premier Schools Exhibition

A high-performance, responsive landing page designed to showcase the **Premier Schools Exhibition**. This project features a sophisticated, modern UI with smooth vertical marquee animations, interactive school categories, and a seamless inquiry registration system.

## 🚀 Live Features

- **Responsive Architecture**: Fully optimized for Desktop, Tablet, and Mobile devices[cite: 1, 2].
- **Dynamic Navigation**: Stateful navbar that transitions elegantly upon scrolling[cite: 3].
- **Vertical Mosaic Carousel**: High-performance CSS-driven vertical image slider.
- **Horizontal Marquee**: Continuous smooth scrolling for participating school logos.
- **Interactive Slider**: Swipeable highlights showcase with custom navigation controls[cite: 2, 3].
- **Lead Capture**: Integrated form structure for parent appointments[cite: 1, 2].

## 🛠 Tech Stack

- **HTML5**: Semantic document structure.
- **CSS3**: Flexbox/Grid layouts with responsive Media Query overrides.
- **JavaScript (ES6+)**: DOM manipulation and scroll-triggered state management[cite: 3].
- **Fonts**: Plus Jakarta Sans (Google Fonts)[cite: 1].
